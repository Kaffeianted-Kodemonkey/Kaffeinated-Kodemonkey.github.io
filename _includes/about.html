What is POUR web design:

  • Perceivable - Users must be able to perceive the information being presented
  • Operable - Users must be able to operate the interface
  • Understandable - Users must be able to understand information & operations
  • Robust - Users must be able to access the content as technology advances

What is ADA compliance, WCAG, Section 508...

ADA compliance Websites adhere to the Americans with Disabilities Act, which requires that websites and digital content be accessible to people with disabilities. The Web Content Accessibility Guidelines (WCAG) and Section 508 are standards that outline guidelines for achieving an ADA compliance website.
